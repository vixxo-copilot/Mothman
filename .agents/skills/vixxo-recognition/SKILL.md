# Vixxo Recognition

## When to use

Use this skill when the user asks to:

- draft recognition messages for team members
- summarize wins by person, team, or initiative
- prepare recognition notes for leadership channels
- convert outcome evidence into specific shout-outs

## Output format

Return in this order:

1. **Recognition candidates** (person/team and why)
2. **Evidence of impact** (what changed, measurable effect if available)
3. **Draft recognition messages** (short, medium, and formal variants)
4. **Send recommendations** (channel, timing, owner)

## Workflow

1. Confirm audience, tone, and channel constraints.
2. Gather recent wins and contributions from available work artifacts.
3. Map each contribution to customer, team, or delivery impact.
4. Draft recognition messages that are specific and evidence-backed.
5. Provide final suggestions for timing and distribution.

## Microsoft 365 guidance

- Use message and meeting context to validate contribution recency.
- Keep generated drafts concise for Teams or email delivery.
- If evidence is thin, ask for one concrete example before finalizing.

## Guardrails

- Avoid generic praise without concrete contribution details.
- Do not fabricate outcomes or metrics.
- Keep content professional and work-context only.
