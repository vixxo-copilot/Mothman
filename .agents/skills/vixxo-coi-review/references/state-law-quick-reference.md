Sample COI 2026 | Not legal advice | Detail: state-law-cancellation-wos.md

## A. 30-day cancellation notice (NOC)

| Situation | Mark | Do |
|-----------|------|-----|
| Endorsement attached (IL 01 03 / CG 02 24) | OK | File endorsement |
| COI wording only | Clarify | Request endorsement |
| No wording, no endorsement | Deficient | Request both |
| Broker cites insured-notice statute | Clarify | Still need cert-holder endorsement |

COI text alone is not enforceable (NY, TX, ACORD 25).

## B. WC waiver of subrogation — tiers

| Tier | States | Do |
|------|--------|-----|
| 1 Prohibited | KY, NH | Escalate; no deficiency |
| 1 Construction | KS, MO, OR | Verify work; escalate |
| 1 Monopolistic | OH, WA, ND, WY | State fund; skip WC WOS |
| 2 Partial | AZ, TX, NJ, MS | Accept if endorsed |
| 2 Partial | NY, ME, VA, WI | Accept if endorsed |
| 2 Partial | PA, CA, NE, LA, CO | Review if high exposure |
| 3 OK | All others | Require WC 00 03 13 |

## C. Tier 2 — WC WOS scope (when endorsed)

| States | Scope |
|--------|-------|
| AZ, TX, NJ, MS | Broad waiver |
| NY, ME, VA | Reimbursement may remain |
| WI | Insurer endorsement required |
| PA, CA, NE | Express waiver for credit |

## D. Endorsement forms

| Line | Form |
|------|------|
| NOC | IL 01 03, CG 02 24 |
| WC WOS | WC 00 03 13; TX: WC 42 03 04 |
| GL WOS | CG 24 04, CG 24 53 |
| Auto WOS | CA 04 44 |

## E. Quick rules

- KY / NH -> no WC WOS
- OH / WA / ND / WY -> state fund only
- MO / OR / KS construction -> escalate first
- NOC on COI only -> get endorsement
- GL or Auto WOS missing -> deficient everywhere
- Escalate: Tier 1 refusal, work type unclear, high-limit Tier 2 SP
