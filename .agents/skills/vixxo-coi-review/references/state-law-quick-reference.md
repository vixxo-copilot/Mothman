# Vixxo COI — State Law Quick Reference

**Sample COI 2026 | Not legal advice | Full analysis: state-law-cancellation-wos.md**

## A. 30-day cancellation notice

| Situation | Mark | Action |
|-----------|------|--------|
| COI + NOC endorsement (IL 01 03, CG 02 24) | Compliant | File endorsement |
| COI language only | Needs clarification | Request NOC endorsement naming Vixxo |
| Neither | Deficient | Request both |
| Broker cites insured-notice statute | Needs clarification | Insured notice ≠ cert holder; still need endorsement |

COI text alone is not enforceable (NY, TX, ACORD 25).

## B. WC WOS — state tiers

| Tier | States | Action |
|------|--------|--------|
| Prohibited | KY, NH | Do NOT deficiency-mark — escalate |
| Construction ban | KS, MO, OR (construction) | Verify work type; escalate |
| Monopolistic fund | OH, WA, ND, WY | State fund cert only; skip WC WOS |
| Partial protection | AZ, TX, NJ, MS, NY, ME, VA, WI, PA, CA, NE, LA, CO | Accept if endorsed; legal review on high exposure |
| Generally OK | All others | Require WC 00 03 13 or state form |

## E. Shortcuts
