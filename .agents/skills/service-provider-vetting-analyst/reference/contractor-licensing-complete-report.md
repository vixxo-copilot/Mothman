## Executive summary

### At a glance

| Metric | Value |
| --- | --- |
| Jurisdictions covered | 56 (50 states + DC + GU, PR, VI, CNMI, AS) |
| Statewide GC license regimes (A) | ~24 states + DC + several territories |
| Trade-only / local GC regimes (B) | ~19 states |
| No statewide GC license (C) | 7 states: CO, IL, IN, MO, NY, PA, WY |
| Dual-state-portal states | OR (CCB + BCD), IA (DIAL + PMSB), MT (DLI + POL), PR (DACO + PCS) |
| Municipal follow-up required | All Regime C states + most Regime B + registration-heavy states |

### Three licensing regimes

| Regime | Meaning | Vetting implication |
| --- | --- | --- |
| **A — Statewide GC** | State board issues general contractor licenses with classifications | Verify state GC portal first; then trade boards for self-performed work |
| **B — Trade-only** | State licenses trades; GC often local or not required statewide | State trade lookup + **city/county** for GC and sometimes trades |
| **C — Local-only GC** | No state GC competency license | **Municipal** portal is primary for GC and most trades |

### Registration is not licensure

These states (and others) require **registration** without a competency exam. An active
registration record does **not** prove trade authorization or municipal permit eligibility.

| State | Registration type | Still verify |
| --- | --- | --- |
| OR | CCB registration (all construction) | BCD trade license for electrical, plumbing, HVAC, etc. |
| IA | Contractor registration | PMSB trade licenses |
| NE | Contractor registration | State electrical + Omaha/Lincoln local trades |
| NJ | HIC registration (NJHIC#) | State trade licenses + municipal |
| PA | HIC registration (residential) | Local GC/trades |
| CT | HIC / NHCC registration | State trade licenses + municipal |
| WA | Contractor registration (L&I) | State trade certifications + municipal |
| RI | CRLB registration | Specialty trade licenses |
| VT | Residential contractor registration | DPS trades |
| MT | Contractor registration (ICEC) | POL trade licenses + municipal |
| AK | Construction contractor registration | Anchorage municipal when applicable |
| PR | DACO contractor registration | Dept. of State trade licenses + municipio |

**Four-layer rule (registration states):** (1) state registration → (2) state trade
license → (3) municipal license → (4) permit eligibility at project address.

### Oregon — dual verification (critical)

| Layer | Portal |
| --- | --- |
| CCB registration (all paid construction) | https://search.ccb.state.or.us/search/ |
| BCD trade license (elec, plumb, HVAC, etc.) | https://www.oregon.gov/bcd/licensing/pages/search.aspx |

Active CCB **does not** authorize trade work. Active BCD **does not** replace CCB.

### Specialty trades summary

GC, plumbing, or electrical licenses **do not** cover these scopes. Verify the **specific**
specialty credential.

| Specialty | Regulator pattern | Common mistake |
| --- | --- | --- |
| **Backflow testing/repair** | Health/env/water utility; TX TCEQ BPAT; many local purveyor lists | Assuming plumber license covers BPAT testing |
| **Fire sprinkler** | State Fire Marshal, contractor board (C-16 in CA) | Confusing with fire **alarm** license |
| **Fire alarm** | SFMO, alarm board, electrical board | Using sprinkler license for alarm work |
| **Pest control** | Dept. of Agriculture (DEC/DEP in NY/NJ) | Company Reg without certified applicator |
| **Locksmith** | BSIS, state police, locksmith board | Locksmith alone for access control/CCTV |
| **Low voltage / CCTV / access** | Alarm board, C-7/SP-FA/LV, limited energy | Standard electrical without alarm/LV class |

**High-signal specialty portals:** TX (TCEQ BPAT + TDI SFMO), CA (BSIS + CSLB C-7/C-16),
FL (CFO SFM + DBPR alarm), NC (NCAGR pest + SP-FA/LV), SC (LLR fire + alarm), WA (DOH BAT + WSDA pest).

### Verification sequence

1. Identify **project state and municipality** (not SP headquarters).
2. Look up **state** registration, GC license, or trade license at official portal.
3. Run **second state portal** if trades split across boards (OR, IA, MT, PR).
4. Look up **municipal** GC/trade license for project address (Regime B/C and Reg states).
5. For specialty scope, verify at **specialty board** (not GC/plumbing/electrical alone).
6. Confirm **Active** status, **classification matches scope**, **name matches contract**.
7. Document lookup date, portal URL, and result (screenshot or saved record).

### Minimum fields to capture

| Field | Why |
| --- | --- |
| License/registration number | Primary lookup key |
| Legal business name | Must match contract and payee |
| Status | Active vs expired, suspended, revoked |
| Classification / endorsement | Scope must match work |
| Expiration date | Lapsed = no permit path |
| Bond / insurance / workers' comp | When shown on board record |
| Disciplinary history | Pattern matters |
| Qualifying party / RMO | Business licensed through a person on many boards |

### Red flags

- License number format wrong for jurisdiction or board
- Name mismatch between board, contract, and invoice (DBA vs legal entity)
- Classification does not cover proposed scope (plumbing ≠ GC; sprinkler ≠ alarm)
- Expired or suspended status with "will renew" explanation only
- Out-of-state license presented for in-state work without reciprocity
- Registration shown where competency license required
- Only GC verified when SP self-performs regulated trades
- Specialty work verified only against general contractor or trade license

### Vetting checklist (SP report)

- [ ] Verify all licenses directly with licensing boards (not SP-provided PDF only)
- [ ] Confirm status, expiration, and disciplinary history
- [ ] Municipal lookup completed when regime is B, C, or registration-heavy
- [ ] Oregon: both CCB and BCD verified when trades involved
- [ ] Specialty credentials verified for backflow, fire, pest, locksmith/LV when in scope
- [ ] Secretary of State entity status checked (entity ≠ contractor license)
- [ ] Document "Not Found" with board searched and date

## Municipal follow-up — registration ≠ licensure

State **registration** (or HIC/NJHIC/DACO consumer registration) confirms
administrative compliance — it is **not** a competency license and often does
**not** authorize trade work. Many of these states also license trades at the
state level; **municipal** licenses, permits, or business certificates may
still be required to pull work locally.

**Vetting rule:** For any **Muni = Yes** row, always run the state registration
lookup **and** the city/county portal for the job site.

## Part 4 — Jurisdiction detail

## How to use this guide

1. Identify the **state or territory where the work will be performed** (not
   where the SP is headquartered).
2. Find the jurisdiction in the [master table](#master-table).
3. Open the **official verification URL** and search by license number and
   legal business name.
4. Confirm **status = Active/Current**, **classification matches scope**, and
   **name matches contract/invoice**.
5. If the regime is **Trade-only** or **Local-only**, run additional lookups
   for trades and the city/county building department.
6. Document findings with lookup date and screenshot or saved record URL.

### Minimum verification fields

| Field | Why it matters |
| --- | --- |
| License/registration number | Primary key for board lookup |
| Legal business name | Must match contract and payment recipient |
| License status | Active vs. expired, suspended, revoked |
| Classification / endorsement | Plumbing license ≠ general building |
| Expiration date | Lapsed licenses block permits and recourse |
| Bond / insurance / workers' comp | Shown on many boards; confirm when available |
| Disciplinary / complaint history | Patterns matter more than isolated old entries |
| Qualifying party / RMO / qualifier | Many boards license the business through a person |

## Licensing regime types

Contractor regulation in the U.S. is **not federal**. Three broad patterns apply:

| Regime | Description | Count (states + DC) | Examples |
| --- | --- | --- | --- |
| **A — Statewide GC license** | State board issues general-contractor licenses (often with classifications) plus separate trade boards | ~24 | CA, AZ, NV, FL, LA, NC, TN, UT, VA |
| **B — Trade-only (state)** | State licenses specific trades (electrical, plumbing, HVAC, roofing); GC handled locally | ~19 | TX, OH, KY, MN, MD, MA, NJ, CT |
| **C — No statewide GC license** | No state-level general contractor competency license; city/county rules apply | 7 states | CO, IL, IN, MO, NY, PA, WY |

**Registration vs. license:** Some states require *registration* (administrative
filing, insurance, workers' comp) without a competency exam — e.g., OR CCB,
IA contractor registration, NE contractor registration, PA/NJ/CT HIC registration.
Registration still must be verified and is legally required, but **does not
replace** trade competency licenses or municipal contractor credentials.

When a state row shows **Reg** or **registration**, assume municipal follow-up
is likely. See [Municipal follow-up — registration ≠ licensure](#municipal-follow-up--registration--licensure).

**Home improvement (HIC) vs. general contractor:** Many states split
residential remodeling/home improvement from commercial GC licensing. Check
both when work is residential or mixed-use.

## Trade licenses (nearly always separate)

Trades are licensed at the **state** level in many jurisdictions, but in
**IL, IN, NY, MO, PA** (and other Regime C states) electrical, plumbing, HVAC,
and GC credentials are often **municipal** — verify at the city/county where
work occurs. See [Municipal licensing — no statewide GC](#municipal-licensing--il-in-ny-mo-pa)
and [Municipal follow-up — registration ≠ licensure](#municipal-follow-up--registration--licensure).

| Trade | Typical licensing agency pattern |
| --- | --- |
| **Electrical** | State board, fire marshal, or **local** building dept. (IL, IN, NY, MO, PA) |
| **Plumbing** | State board (IN: statewide; IL: state; NY/MO/PA: often **local**) |
| **HVAC / mechanical / refrigeration** | State mechanical board or **local** (especially IL, IN, MO, PA) |
| **Roofing** | State board (IL: IDFPR), AG registration, or **local** |
| **Asbestos / lead / abatement** | EPA + state environmental or labor agency |
| **Fire protection (sprinkler / alarm / suppression)** | State fire marshal, contractor board, or **local** — sprinkler and alarm are often **separate** licenses |
| **Backflow prevention (testing / repair)** | Health dept, env quality, water utility, or plumbing board add-on — often **local water purveyor** lists |
| **Pest control (structural / commercial applicator)** | State Dept. of Agriculture (DEC/DEP in NY/NJ) — company + applicator credentials |
| **Locksmith / low voltage / security alarm** | BSIS, alarm board, limited-energy electrical, or **local** — access control/CCTV usually **not** covered by locksmith alone |

For commercial facilities work (Vixxo SP context), verify **both** the GC
credential (state or local) **and** every trade license for self-performed work.

**Specialty trades detail:** See [contractor-licensing-specialty-trades.md](contractor-licensing-specialty-trades.md) for per-state verification URLs for backflow, fire protection, pest control, and locksmith/low voltage/alarm credentials.

### Oregon — CCB registration AND BCD trade licenses

Oregon requires **two independent credentials** for most construction work:

| Layer | Authority | Verification URL |
| --- | --- | --- |
| **CCB registration** | Construction Contractors Board | https://search.ccb.state.or.us/search/ |
| **BCD trade license** | Building Codes Division (DCBS) | https://www.oregon.gov/bcd/licensing/pages/search.aspx |

- **CCB** is required for virtually all paid construction (residential and
  commercial) — registration with bond, insurance, and endorsements.
- **BCD** separately licenses electrical, plumbing, boiler, elevator,
  manufactured dwelling, and mechanical/HVAC trades.
- An active CCB record does **not** prove trade authorization. An active BCD
  trade license does **not** substitute for CCB registration. Verify **both**
  portals when trade work is involved.
- Official guidance: https://www.oregon.gov/bcd/lbdd/pages/licensed-work.aspx

## Municipal follow-up — registration ≠ licensure

These states issue **registration** (or consumer-protection registration) at
the state level. Registration confirms administrative compliance — it is **not**
equivalent to a competency license and typically does **not** authorize trade
work on its own. Most also license trades separately at the state level; many
cities and counties add **local** contractor, trade, or business licenses.

**Compact table:** [contractor-licensing-quick-lookup-table.md](contractor-licensing-quick-lookup-table.md) — "Municipal follow-up" section.

| --- | --- | --- | --- |
| **AK** | https://www.commerce.alaska.gov/cbp/main/Search/Professional | State Reg classification | Anchorage: https://bsd.muni.org/contractorlicensing/search.aspx |
| **CT** | https://www.elicense.ct.gov/Lookup/LicenseLookup.aspx | HIC/NHCC Reg + **state trade licenses** | Town ROW licenses (local engineering dept) |

4. **Permit** eligibility in that jurisdiction

Anchorage, Boise, Newark, Jersey City, Austin, Dallas, San Antonio, etc.).

## Local jurisdiction checklist

Use when the state row shows **Regime C**, **Trade-only**, **"local check"**,
**registration (Reg)**, or for **IL, IN, NY, MO, PA** regardless of trade:

1. Identify the **city and county** of the job site.
2. Run the **state registration** lookup (if applicable) — confirm it is not mistaken for a trade license.
3. Run the **state trade license** lookup on the correct board (may be a different portal).
4. Search `"[city/county] contractor license lookup"` or use municipal URLs in the [quick lookup table](contractor-licensing-quick-lookup-table.md).
5. Confirm whether a **local GC license**, **trade license**, **supervisor certificate**, or **business tax receipt** is required.
6. Notable local licensing cities: **Chicago (IL)**, **NYC (NY)**, **Philadelphia (PA)**, **Pittsburgh (PA)**, **Indianapolis (IN)**, **Omaha/Lincoln (NE)**, **Anchorage (AK)**, **Kansas City / St. Louis (MO)**, **Austin/Dallas/San Antonio (TX)**, **Denver (CO)**.
7. For **Oregon**, verify **both** CCB and BCD regardless of municipality.

## Reciprocity and multi-state work

- **No universal reciprocity.** A license in one state rarely authorizes work in another.
- **Partial reciprocity** exists between some neighboring states for specific classifications (e.g., NC ↔ TN ↔ GA in limited cases).
- SPs working across state lines typically need **separate licenses** in each state where work is performed.
- Always verify the receiving state's requirements directly with the licensing board.

## Red flags during verification

| Signal | Action |
| --- | --- |
| No license number when one is required | Stop — request number; verify before proceeding |
| Status expired, suspended, or revoked | Hard stop — do not engage |
| Business name mismatch (license vs. contract) | Investigate entity structure; may be subcontractor issue |
| Classification doesn't cover scope | Require correct license or licensed subcontractor |
| Refusal to provide license number | Treat as disqualifying for licensed work |
| Only business registration (SOS) found, no contractor license | SOS ≠ contractor authorization — keep searching |
| Out-of-state license offered for in-state work | Invalid unless reciprocity confirmed in writing with board |
