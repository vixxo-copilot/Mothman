# Specialty Trade Licensing — Backflow, Fire, Pest Control, Locksmith / Low Voltage

Companion to [contractor-licensing-us-states-territories.md](contractor-licensing-us-states-territories.md).

**Last reviewed:** 2026-06-30

Covers **backflow prevention**, **fire protection** (sprinkler / alarm / suppression),
**pest control** (structural and commercial applicator), and **locksmith / low voltage /
security alarm** credentials — often licensed on **separate boards** from general
contractor or standard electrical/plumbing licenses.

> **Vetting rule:** A valid GC, plumbing, or electrical license does **not** prove
> authorization for these specialties. Verify the **specific credential** for the work
> scope at the **official agency** for the project state (and municipality, if applicable).

---

## Quick verification checklist

| Specialty | Confirm on record | Common mistake |
| --- | --- | --- |
| **Backflow** | Tester certification / BPAT license; ASSE/NICET cert current | Assuming plumber license covers backflow **testing** |
| **Fire — sprinkler** | Fire protection / sprinkler contractor license | Confusing with fire **alarm** or general plumbing |
| **Fire — alarm** | Fire alarm / burglar alarm / low-voltage alarm license | Confusing sprinkler license with alarm monitoring/install |
| **Pest control** | Structural pest control company + applicator/technician credentials | Confusing business Reg with certified applicator license |
| **Locksmith** | Locksmith company/employee registration (where required) | Lock work without separate alarm/low-voltage license when access control is involved |
| **Low voltage / CCTV / access** | Alarm contractor, limited energy, C-7/low-voltage classification | Standard electrical license may not cover security/CCTV (state-specific) |

## Backflow prevention

**What it covers:** Testing, repair, and installation of backflow prevention assemblies
on domestic, irrigation, fire, and commercial water lines.

**Regulatory pattern:**

| Pattern | Where | Verification approach |
| --- | --- | --- |
| **State environmental / health certification** | TX, WA, CT, MN, NY | State tester certification list or license portal |
| **Plumbing board add-on certification** | MN, some others | Same portal as plumbing + backflow cert type (BT/BF) |
| **Local water purveyor / county health list** | Many states | Water utility approved-tester roster (no single state URL) |
| **Fire-line backflow** | TX and others | May require affiliation with approved fire-line contractor |

**National references:** American Backflow Prevention Association (ABPA); ASSE Series 5110
tester certification (required or recognized in many states).

## Fire protection (sprinkler, alarm, suppression)

**What it covers:**

- **Sprinkler / water-based:** Installation, inspection, and service of automatic sprinkler systems
- **Special hazards / suppression:** Kitchen hood, clean agent, dry chemical systems
- **Fire alarm / detection:** Install, monitor, service fire alarm and often burglar alarm systems
- **Fire equipment:** Portable extinguishers, hood systems (sometimes separate dealer license)

**Regulatory pattern:**

| Pattern | Examples | Notes |
| --- | --- | --- |
| **State Fire Marshal (SFMO)** | TX, FL, LA, NV | Company + individual licenses; often separate sprinkler vs alarm |
| **Contractor licensing board** | SC LLR, NC, TN, CA CSLB | Fire sprinkler as classification; alarm may be separate board |
| **Construction / CIB board** | OK CIB, IA DIAL | Fire extinguishing + alarm programs |
| **Electrical / alarm board** | FL DBPR Electrical, GA | Alarm work under electrical contractor licensing |
| **Municipal** | NYC, Chicago, Philadelphia | Fire alarm often on L&I trade license (see municipal tables) |

## Pest control (structural pest & commercial applicator)

**What it covers:** Termite/WDO treatment, rodent control, general structural pest control,
and application of pesticides in and around buildings (commercial applicator categories).

**Regulatory pattern:** Nearly always the state **Department of Agriculture** (or
**Environmental Conservation / DEP** in NY, NJ). Separate from contractor GC licensing.

**Categories to verify:**

| Credential | Who holds it |
| --- | --- |
| **Company license** | Pest control business operating in state |
| **Certified applicator / operator** | Supervisory credential |
| **Registered technician** | Technician working under supervisor |
| **Commercial applicator category** | Specific pest types (structural, WDO, fumigation, etc.) |

**National reference:** Association of American Pesticide Control Officials (AAPCO) maintains
state agency contacts — https://aapco.org/

Many states use **Kelly Solutions** for online applicator lookup:
`https://www.kellysolutions.com/{state}/Applicators/` (not all states).

## Locksmith, low voltage, security alarm & CCTV

**What it covers:**

- **Locksmith:** Locks, safes, rekeying, non-electronic access hardware
- **Low voltage / limited energy:** CCTV, access control, structured cabling (often ≤50–91V)
- **Security / fire alarm:** Burglar alarm, fire alarm monitoring and installation
- **Electronic access control:** Often spans locksmith + alarm + low-voltage licenses

**Regulatory pattern:**

| Pattern | States / examples |
| --- | --- |
| **Dedicated locksmith license** | CA, TX, NC, VA, MD, NV, NJ, OK, OR, AL, LA, IL (sunset 2029) |
| **Alarm / security contractor license** | FL (DBPR electrical/alarm), SC (LLR), TX (DPS/private security + alarm), OK (CIB) |
| **Low-voltage electrical classification** | CA C-7, NC SP-FA/LV, UT low voltage, TN alarm licenses |
| **Electronic Security Board** | AL (AESBL), AR (State Police alarm) |
| **Exempt under electrical (limited)** | CO, GA, AR (partial), HI (partial) — verify scope |
| **Municipal only** | Many cities when no state license |

**Important:** Installing **electronic access control, CCTV, or fire alarm** usually
requires credentials **beyond** a locksmith license. Verify alarm/low-voltage/fire alarm
classification separately.

## Cross-specialty vetting scenarios (Vixxo SP context)

| Work scope | Minimum credentials to verify |
| --- | --- |
| Backflow test at commercial site | State/local BPAT or certified tester + (if repair) plumbing/backflow cert |
| Fire sprinkler inspection/repair | State fire sprinkler contractor license + any CCB/GC/local GC |
| Fire alarm panel service | Fire alarm contractor license (not sprinkler) + low-voltage where required |
| Rodent / termite treatment | Structural pest control company license + certified applicator |
| Door hardware rekey only | Locksmith (if state requires) |
| Access control / CCTV install | Alarm/low-voltage/electrical — **not** locksmith alone |
| Integrated fire + security | Multiple licenses across fire alarm, low voltage, and possibly sprinkler boards |

## Integration with main vetting workflow

When a service provider performs specialty work, add these checks to the Manual Review Checklist:

- [ ] Confirm specialty credential type matches scope (backflow vs fire sprinkler vs alarm vs pest vs locksmith/LV).
- [ ] Verify at the **specialty board portal**, not only GC/plumbing/electrical.
- [ ] For fire alarm work, confirm **alarm** license — not sprinkler-only.
- [ ] For pest work, confirm **company license** and **applicator/technician** credentials.
- [ ] For access control/CCTV, confirm **low-voltage/alarm** license — not locksmith-only.
- [ ] Check municipal trade license if state registration is administrative only.
